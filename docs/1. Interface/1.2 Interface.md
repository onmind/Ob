# Interface 

## 인터페이스 

블렌더의 인터페이스는 크게 3가지로 구분됩니다. 
- 가장 위쪽에 있는 top bar 
- 중간에 있는 영역패널 Area 
- 가장 하단에 자리잡은 상태표시줄 패널 Status Bar 

## Topbar 
- 톱바라고 부르는 이곳은 주로 메뉴가 있는 곳입니다. 
- 기본적인 File, Edit, Render, Window 그리고 도움말인 help가 있습니다.
